# C++ Bank
### v1.2
Sistema para un banco en C++

[GitHub Repo](https://github.com/MateoDi9z/cppbank)

## Features
  - [x] Sistema de Menus
  - [x] Crear/Ver/Borrar Usuario
  - [ ] Crear/Ver/Borrar Transacciones
  - [ ] Calcular balances en base a Transacciones
  - [ ] Modificar Balances
  - More...

## Requerimientos
  - './db/TxDB.txt' 
  - './db/UserDB.txt'

> Build main.cpp
```sh
g++ main.cpp
```

> Run a.exe
```sh
a.exe
```

> Build & run in powershell
```sh
.\cmd\.ps1
```
